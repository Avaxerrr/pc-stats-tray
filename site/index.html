Deployment Protocol

Quick Setup

  1. Install the .NET 10 Desktop Runtime.
  2. Run PCStatsTray.exe.
  3. Right-click the tray icon and open OSD Settings.
  4. Turn on Desktop OSD for desktop metrics.
  5. Turn on RTSS OSD for in-game output if RTSS is installed and running.

Architecture

The RTSS Bridge

PC Stats Tray collects the hardware data, formats the text, and decides what should be displayed. RTSS (RivaTuner Statistics Server) is the renderer that draws those stats inside supported games.

To enable the in-game overlay, make sure RTSS is running in the background and Show On-Screen Display is enabled in its profile.
