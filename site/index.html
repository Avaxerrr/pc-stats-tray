PC Stats Tray

A lightweight Windows app that shows PC stats in the tray, on the desktop, and inside games through RTSS.

  • Small tray icon with live temperature
  • Desktop overlay with font, shadow, outline, and layout controls
  • Optional RTSS support for in-game on-screen display

What you need

Desktop only: the app plus the .NET 10 Desktop Runtime.

In-game overlay: the app, the .NET 10 Desktop Runtime, and RTSS.

Download .NET 10 Download RTSS

Quick setup

  1. Install the .NET 10 Desktop Runtime.
  2. Run PCStatsTray.exe.
  3. Right-click the tray icon and open OSD Settings.
  4. Turn on Desktop OSD for the desktop overlay.
  5. Turn on RTSS OSD (Games) only if RTSS is installed and running.

How RTSS fits in

PC Stats Tray collects and formats the hardware data. RTSS is the separate tool that draws that text inside supported games.

If you want the in-game overlay, open RTSS and make sure Show On-Screen Display is enabled.
