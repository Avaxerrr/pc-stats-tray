PC Hardware
Monitoring.

PC Stats Tray is a lightweight Windows utility that keeps your system vitals visible where they matter: in the tray icon, on the desktop, or inside games through RTSS.

  • Live CPU and GPU temperatures in the tray icon
  • Desktop OSD with font, shadow, outline, and layout controls
  • Optional RTSS output for supported in-game overlays

Deployment Protocol

Quick Setup

  1. Install the .NET 10 Desktop Runtime.
  2. Run PCStatsTray.exe.
  3. Right-click the tray icon and open OSD Settings.
  4. Turn on Desktop OSD for the desktop overlay.
  5. Turn on RTSS OSD (Games) only if RTSS is installed and running.

Architecture

The RTSS Bridge

PC Stats Tray handles the hardware polling, metric formatting, and layout choices. RTSS is the renderer that draws those metrics inside supported games.

For the in-game overlay to appear, RTSS must be running with Show On-Screen Display enabled in its profile.
